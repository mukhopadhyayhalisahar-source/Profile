# **App Name**: Neural Canvas

## Core Features:

- Immersive Terminal Intro: A full-screen, unskippable terminal boot sequence to greet website visitors with contextual information and set a high-tech tone.
- Dynamic Hero Section: An interactive landing page utilizing a WebcamPixelGridDemo to present the founder's vision and professional scope with compelling visuals and calls to action.
- AI-Driven Project Insights: A tool to generate concise, AI-powered summaries or highlight key innovations for complex AI projects like 'Lynex' to enhance visitor comprehension and engagement on the website.
- Project Lynex Showcase: A detailed interactive display section for Project Lynex, an 'Intelligent Personal AI Assistant', incorporating ZoomParallax for imagery and an embedded SplineScene for 3D visualizations.
- Project Gameox Presentation: An engaging presentation section for Project Gameox, a 'Gesture-Operated Health Analytics Game,' using an animated MacbookScroll for visual storytelling and project details.
- Personal & Professional Milestones: A structured and engaging timeline section displaying the founder's medical education, technical skills, and independent projects, utilizing StickyScroll for smooth progression.
- Contact & Collaboration: A dedicated section for contact information and links to professional profiles (GitHub, LinkedIn), fostering collaboration and networking opportunities through the website.

## Style Guidelines:

- Color scheme: Dark theme, embodying sophistication and technology suitable for a high-end portfolio website.
- Primary color: A rich, deep indigo-purple (#5A46B9), chosen to convey expertise and a high-end, futuristic aesthetic against a dark background.
- Background color: A very dark, subtle purplish-black (#121117), providing depth and complementing the primary color without being overpowering.
- Accent color: A clear, vibrant medium blue (#6791E4), used for highlights, interactive elements, and calls to action, ensuring strong contrast and a dynamic feel.
- Headline font: 'Space Grotesk' (sans-serif), for a modern, techy, and scientific aesthetic suitable for titles on a portfolio website.
- Body font: 'Inter' (sans-serif), providing a clean, objective, and highly readable typeface for detailed descriptions and general content.
- Utilize modern, clean, and minimalist vector icons (e.g., Lucide React icons) in a consistent style to visually represent key concepts and sections across the website.
- An interactive, multi-section layout transitioning between full-screen elements, parallax effects, scroll-based content reveals, and embedded 3D models for an engaging user journey on the website.
- Smooth transitions, subtle motion effects, and dynamic scroll-based animations (leveraging Framer Motion) to create an fluid and responsive user experience for the website.
